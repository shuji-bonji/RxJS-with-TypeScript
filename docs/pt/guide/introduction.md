# Introdução

Este site de documentação fornece conteúdo voltado para "Aprender RxJS para Programadores TypeScript".

## Primeiros Passos

Para começar seu projeto, siga estes passos simples:

1. Para instruções sobre como configurar seu ambiente de desenvolvimento para aprendizado prático, consulte [esta página](starter-kid.md).
2. Para informações detalhadas sobre RxJS, visite o [site oficial do RxJS](https://rxjs.dev).
3. Informações mais detalhadas também podem ser encontradas na [referência oficial da API do RxJS](https://rxjs.dev/api).
