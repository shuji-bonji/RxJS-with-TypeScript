# Apêndice

Esta seção descreve aplicações práticas do RxJS e como usar o RxJS em campos específicos.

O objetivo é fornecer tópicos mais especializados e aplicações práticas não abordados no guia principal para aprofundar ainda mais a compreensão do RxJS.

| Tópico | Visão Geral | Público-Alvo |
|---------|------|---------|
| [Desenvolvimento Embarcado e Programação Reativa](/pt/guide/appendix/embedded-reactive-programming) | Utilização do RxJS em sistemas embarcados, integração de sensores, detecção de correlação de eventos | Desenvolvedores embarcados, engenheiros de IoT |
| [Abordagens Reativas Além do ReactiveX](/pt/guide/appendix/reactive-patterns-beyond-rxjs) | Sete abordagens incluindo orientado a eventos, máquinas de estado e programação de fluxo de dados | Engenheiros de controle, desenvolvedores embarcados |
| [Mapa Abrangente de Arquitetura Reativa](/pt/guide/appendix/reactive-architecture-map) | Estrutura de 7 camadas de UI, comunicação, backend, pipeline de dados, IoT, controle e UX | Engenheiros full-stack, arquitetos |
| [Programação Reativa Reconsiderada](/pt/guide/appendix/reactive-programming-reconsidered) | Lacuna entre ideais e realidade da PR, pontos fortes e limitações, como identificar áreas de aplicação | Todos os desenvolvedores, arquitetos |
