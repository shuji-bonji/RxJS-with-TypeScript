# Introduction

Ce site de documentation fournit du contenu destiné à "Apprendre RxJS pour les programmeurs TypeScript".

## Démarrage

Pour commencer votre projet, suivez ces étapes simples :

1. Pour obtenir des instructions sur la configuration de votre environnement de développement en vue d'un apprentissage pratique, voir [cette page](starter-kid.md).
2. Pour des informations détaillées sur RxJS, visitez le [site officiel de RxJS](https://rxjs.dev).
3. Des informations plus détaillées sont également disponibles dans la [référence API officielle de RxJS](https://rxjs.dev/api).
