# Introduction

This documentation site provides content aimed at "Learning RxJS for TypeScript Programmers."

## Getting Started

To begin your project, follow these simple steps:

1. For instructions on setting up your development environment for hands-on learning, see [this page](starter-kid.md).
2. For detailed information about RxJS, visit the [RxJS official website](https://rxjs.dev).
3. More detailed information can also be found in the [RxJS official API reference](https://rxjs.dev/api).
