# Appendice

Questa sezione spiega esempi pratici di applicazione di RxJS e metodi di utilizzo in settori specifici.

L'obiettivo è fornire argomenti più specializzati ed esempi di applicazione pratica non coperti nella guida principale, approfondendo ulteriormente la comprensione di RxJS.

## Contenuti dell'appendice

| Argomento | Panoramica | Lettori target |
|---------|------|---------|
| [Sviluppo embedded e programmazione reattiva](/it/guide/appendix/embedded-reactive-programming) | Utilizzo di RxJS nei sistemi embedded, integrazione di sensori, rilevamento di correlazioni tra eventi | Sviluppatori embedded, ingegneri IoT |
| [Approcci reattivi oltre ReactiveX](/it/guide/appendix/reactive-patterns-beyond-rxjs) | Sette approcci tra cui event-driven, state machine, data flow programming | Ingegneri di controllo, sviluppatori embedded |
| [Mappa completa dell'architettura reattiva](/it/guide/appendix/reactive-architecture-map) | Struttura a 7 livelli: UI, comunicazione, backend, data pipeline, IoT, controllo, UX | Ingegneri full-stack, architetti |
| [Reactive Programming Reconsidered](/it/guide/appendix/reactive-programming-reconsidered) | Gap tra ideale e realtà di RP, punti di forza e limiti, come identificare i settori di applicazione | Tutti gli sviluppatori, architetti |
| [RxJS e l'ecosistema Reactive Streams](/it/guide/appendix/rxjs-and-reactive-streams-ecosystem) | Quadro generale dello stack tecnologico come RxJS, Reactor, Kafka e coordinamento tra UI e livello dati | Ingegneri full-stack, ingegneri backend |
