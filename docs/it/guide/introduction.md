# Introduzione

Questo sito di documentazione fornisce contenuti finalizzati all'"apprendimento di RxJS per programmatori TypeScript".

## Come iniziare

Per iniziare con il progetto, seguite questi semplici passaggi:

1. Per informazioni sulla configurazione dell'ambiente di sviluppo per l'apprendimento pratico, consultate [questa pagina](starter-kid.md).
2. Per saperne di più su RxJS, visitate il [sito ufficiale di RxJS](https://rxjs.dev).
3. Per informazioni più dettagliate, consultate il [riferimento API sul sito ufficiale di RxJS](https://rxjs.dev/api).
