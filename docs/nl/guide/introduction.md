# Inleiding

Deze documentatiesite biedt inhoud met als doel "RxJS leren voor TypeScript programmeurs".

## Hoe te beginnen

Om met het project te beginnen, volgt u deze eenvoudige stappen:

1. Voor instructies over het opzetten van een ontwikkelomgeving voor hands-on leren, raadpleeg [hier](starter-kid.md).
2. Als u meer wilt weten over RxJS, zie de [officiële RxJS website](https://rxjs.dev).
3. Voor meer gedetailleerde informatie kunt u ook de [API referentie op de officiële RxJS website](https://rxjs.dev/api) raadplegen.
