# Bijlage

Deze sectie behandelt praktische toepassingsvoorbeelden van RxJS en gebruiksmethoden in specifieke domeinen.

Het doel is om uw begrip van RxJS verder te verdiepen door meer gespecialiseerde onderwerpen en praktische toepassingsvoorbeelden te bieden die niet in de hoofdgids zijn behandeld.

## Inhoud van de bijlage

| Onderwerp | Overzicht | Doelgroep |
|---------|------|---------|
| [Embedded Development en Reactief Programmeren](/nl/guide/appendix/embedded-reactive-programming) | RxJS-gebruik in embedded systemen, sensorintegratie, gebeurteniscorrelatie-detectie | Embedded developers, IoT-engineers |
| [Reactieve Methoden Buiten ReactiveX](/nl/guide/appendix/reactive-patterns-beyond-rxjs) | 7 methoden zoals event-driven, state machines, dataflow programming | Control engineering engineers, embedded developers |
| [Reactieve Architectuur Totaalkaart](/nl/guide/appendix/reactive-architecture-map) | 7-laags structuur: UI, communicatie, backend, data pipeline, IoT, control, UX | Fullstack engineers, architecten |
| [Reactive Programming Reconsidered](/nl/guide/appendix/reactive-programming-reconsidered) | Kloof tussen RP ideaal en realiteit, sterke punten en beperkingen, hoe toepassingsgebieden te bepalen | Alle developers, architecten |
| [RxJS en Reactive Streams Ecosysteem](/nl/guide/appendix/rxjs-and-reactive-streams-ecosystem) | Totaaloverzicht van technology stack zoals RxJS, Reactor, Kafka en integratie van UI tot data layer | Fullstack engineers, backend engineers |

## Toekomstige toevoegingen gepland

De bijlage sectie zal in de toekomst verder worden uitgebreid. We overwegen de volgende onderwerpen:

- Praktische patronen voor WebSocket-integratie
- RxJS-gebruik in PWA's
- Integratie met Web Components
- RxJS op server-side (Node.js)
- Patronen voor realtime data processing

Voor verzoeken en feedback kunt u terecht op [GitHub Discussions](https://github.com/shuji-bonji/RxJS-with-TypeScript/discussions).
