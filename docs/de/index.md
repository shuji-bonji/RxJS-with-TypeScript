Dies ist eine Seite zum systematischen Erlernen von RxJS in der TypeScript-Umgebung.
Verstehen Sie es intuitiv und logisch durch praktischen Code und Tests.


> [!IMPORTANT]
> Dieses "RxJS with TypeScript"-Projekt ([Github](https://github.com/shuji-bonji/RxJS-with-TypeScript)) wurde in Zusammenarbeit zwischen menschlichen Ingenieuren ([@shuji-bonji](https://github.com/shuji-bonji)) und generativer KI ([ChatGPT](https://chatgpt.com/), [Claude](https://claude.ai/)) entwickelt.
> Dieses Projekt soll ein Modellfall für "von Menschen und KI gemeinsam erstellte Bildungsmaterialien" sein.

> [!NOTE]
> - [RxJS (Reactive Extensions for JavaScript)](https://rxjs.dev) ist eine Bibliothek, die asynchrone Programmierung und ereignisbasierte Programmierung unterstützt.
> - [TypeScript](https://www.typescriptlang.org) ist eine stark typisierte Programmiersprache, die auf JavaScript basiert.
> - Diese deutsche Version wurde mit [Claude Code](https://claude.ai/code) und [DeepL MCP Server](https://github.com/modelcontextprotocol/servers/tree/main/src/deepl) aus dem Japanischen übersetzt.
