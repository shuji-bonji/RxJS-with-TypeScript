# Einführung

Diese Dokumentationsseite bietet Inhalte zum "Erlernen von RxJS für TypeScript-Programmierer".

## Erste Schritte

Um mit Ihrem Projekt zu beginnen, folgen Sie diesen einfachen Schritten:

1. Lernen Sie, wie man eine Entwicklungsumgebung für praktisches Lernen erstellt [hier](/de/guide/starter-kid.md).
2. Wenn Sie mehr über RxJS erfahren möchten, besuchen Sie bitte die [offizielle RxJS-Website](https://rxjs.dev).
3. Weitere Informationen finden Sie auch unter [API-Referenz auf der offiziellen RxJS-Website](https://rxjs.dev/api).
