Este es un sitio para aprender RxJS sistemáticamente en el entorno TypeScript.
Compréndelo de forma intuitiva y lógica a través de código práctico y pruebas.


> [!IMPORTANT]
> Este proyecto "RxJS con TypeScript" ([Github](https://github.com/shuji-bonji/RxJS-with-TypeScript)) se crea mediante la colaboración entre ingenieros humanos ([@shuji-bonji](https://github.com/shuji-bonji)) e IA generativa ([ChatGPT](https://chatgpt.com/), [Claude](https://claude.ai/)).
> Este proyecto pretende ser un caso modelo de "materiales educativos co-creados por humanos e IA".

> [!NOTE]
> - [RxJS (Reactive Extensions for JavaScript)](https://rxjs.dev) es una biblioteca que soporta la programación asíncrona y la programación basada en eventos.
> - [TypeScript](https://www.typescriptlang.org) es un lenguaje de programación fuertemente tipado basado en JavaScript.
> - Esta versión en español se tradujo del japonés utilizando [Claude Code](https://claude.ai/code) y [DeepL MCP Server](https://github.com/DeepLcom/deepl-mcp-server).
