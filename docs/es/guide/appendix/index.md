# Apéndice

Esta sección describe las aplicaciones prácticas de RxJS y cómo utilizar RxJS en campos específicos.

El objetivo es proporcionar temas más especializados y aplicaciones prácticas que no se cubren en la guía principal para profundizar aún más en la comprensión de RxJS.

## Contenidos del apéndice

| Tema | Descripción general | Público objetivo |
|---------|------|---------|
| [Desarrollo embebido y programación reactiva](/es/guide/appendix/embedded-reactive-programming) | Utilización de RxJS en sistemas embebidos, integración de sensores, detección de correlación de eventos | Desarrolladores embebidos, ingenieros de IoT |
| [Enfoques reactivos más allá de ReactiveX](/es/guide/appendix/reactive-patterns-beyond-rxjs) | Siete enfoques incluyendo arquitectura dirigida por eventos, máquinas de estado y programación de flujo de datos | Ingenieros de control, desarrolladores embebidos |
| [Mapa integral de arquitectura reactiva](/es/guide/appendix/reactive-architecture-map) | Estructura de 7 capas: UI, comunicación, backend, pipeline de datos, IoT, control y UX | Ingenieros full-stack, arquitectos |
| [Reconsiderando la programación reactiva](/es/guide/appendix/reactive-programming-reconsidered) | Brecha entre los ideales y la realidad de RP, fortalezas y limitaciones, cómo identificar áreas de aplicación | Todos los desarrolladores, arquitectos |
| [RxJS y el ecosistema de Reactive Streams](/es/guide/appendix/rxjs-and-reactive-streams-ecosystem) | Stack tecnológico completo incluyendo RxJS, Reactor, Kafka, e integración de UI a capa de datos | Ingenieros full-stack, ingenieros backend |
