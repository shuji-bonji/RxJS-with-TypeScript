# Introducción

Este sitio de documentación proporciona contenido dirigido a "Aprender RxJS para Programadores TypeScript."

## Comenzando

Para comenzar tu proyecto, sigue estos sencillos pasos:

1. Para instrucciones sobre cómo configurar tu entorno de desarrollo para aprendizaje práctico, consulta [esta página](starter-kid.md).
2. Para información detallada sobre RxJS, visita el [sitio web oficial de RxJS](https://rxjs.dev).
3. También puedes encontrar información más detallada en la [referencia de API oficial de RxJS](https://rxjs.dev/api).
